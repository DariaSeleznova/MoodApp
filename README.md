
# 🎬 MoodApp

MoodApp is a web application that helps you discover movies, TV shows, and music based on your current mood.

Instead of endlessly scrolling, just choose how you feel — and get personalized recommendations.

## ✨ Features

- 🎭 Mood-based recommendations
- 🎬 Movies & TV shows via TMDB API
- 🎧 Music suggestions via Last.fm
- ❤️ Favorites system
- 🌍 Multi-language support
- 📱 Responsive design

## 🛠 Tech Stack

- JavaScript (ES6+)
- Webpack
- SCSS
- Firebase (Authentication & Database)
- TMDB API
- Last.fm API

## ⚙️ Installation
git clone https://github.com/DariaSeleznova/MoodApp.git
cd MoodApp
npm install
npm start
## 🔑 Environment Variables

## Create a .env file:

- TMDB_API_KEY=your_key
- LASTFM_API_KEY=your_key
- FIREBASE_API_KEY=your_key
